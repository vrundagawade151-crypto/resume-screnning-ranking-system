# Resume Screening & Ranking System

## Description
A web-based system that screens and ranks resumes based on job requirements.

## Technologies
- Frontend: HTML, CSS, JavaScript
- Backend: Python (Flask)
- Database: MySQL

## Features
- Job Posting
- Resume Upload
- Skill Matching
- Automatic Ranking
- Shortlisting

## How to Run

### 1. Start MySQL
Create database using sql/database.sql

### 2. Install Backend
cd backend
pip install -r requirements.txt

### 3. Run Backend
python app.py

### 4. Open Frontend
Open frontend/index.html in browser
